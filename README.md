# Aday Takip v4.4

## Çalıştırma

```bash
cd ~/Downloads && rm -rf aday_takip_meslek_excel_sistemi_v4_4 && unzip aday_takip_meslek_excel_sistemi_v4_4.zip && cd aday_takip_meslek_excel_sistemi_v4_4 && python3 -m streamlit run app.py --server.port 8511
```

Tarayıcı:

```text
http://localhost:8511
```

## İlk kullanım

1. Ayarlar > Veritabanını sıfırla
2. Excel Kaynak Yükleme > Paket içindeki Excel’i içe aktar
3. Tanımlar ve Ücretler > Değerlendirici/Gözetmen
4. Aday Kaydet
5. Sınav Planlama
6. Sınav Takvimi
